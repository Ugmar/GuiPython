# GuiPython

Репозиторий содержит лабораторные работы по созданию приложений с графическим интерфейсом (GUI) и интерактивных программ, выполненные в рамках обучения на 1 курсе (2 семестр). Автор: Марчуков Антон Игоревич (Группа ИУ7-25Б).

## 📋 Описание проекта

Проект представляет собой комплекс программ, демонстрирующих навыки проектирования GUI, обработки графических данных, работы со стеганографией, численной геометрией, а также создания двумерной анимации. Каждая лабораторная работа содержит основную реализацию, а также защитные модули (`def/`), подтверждающие глубокое понимание алгоритмической базы.

---

## 🛠 Используемые технологии

В проекте применяются как классические инструменты построения интерфейсов, так и специализированные мультимедийные библиотеки:

* **Язык программирования:** Python 3.x
* **Графические интерфейсы и библиотеки визуализации:**
    * `Tkinter` — основной фреймворк для построения оконных приложений, работы с виджетами, валидации ввода и кастомной отрисовки через `Canvas`.
    * `Pygame` — библиотека для разработки мультимедийных приложений, использованная для построения игрового цикла, рендеринга 2D-анимации, обработки событий клавиатуры/мыши и работы с частотой кадров (FPS).
* **Математика и алгоритмы:**
    * `math` (`sin`, `cos`, `radians`, `sqrt`, `atan2`) — тригонометрические вычисления для геометрии и физики движения объектов.
    * `time` — управление временными интервалами и замеры производительности.
* **Работа с графическими файлами:**
    * Побайтовая обработка изображений в формате `.bmp` для низкоуровневой работы с пикселями.

## 📂 Структура проекта и описание лабораторных работ

### `lab1/` — Калькулятор систем счисления
* **Что делает:** Оконное приложение на `Tkinter`, представляющее собой специализированный калькулятор. Он позволяет выполнять арифметические операции (сложение, вычитание) над числами в кастомных системах счисления и переводить результаты между ними.
* **Особенности:** Присутствует меню управления, обработка некорректного ввода пользователя, очистка полей и вывод детальных ошибок.

### `lab2/` — Численные методы и построение графиков функций
* **Что делает:** Приложение для поиска корней нелинейных уравнений различными численными методами (например, метод половинного деления, метод хорд или Ньютона) на заданном интервале.
* **Особенности:** Визуализация графика функции, экстремумов, точек перегиба и построение таблицы итераций с оценкой погрешности.

### `lab3/` — Стеганография (Скрытие данных в изображениях)
* **Что делает:** Программа для скрытия текстовой информации внутри графического файла формата `BMP` (метод LSB — Least Significant Bit / Метод наименьшего значащего бита) и последующего её извлечения.
* **Архитектура модулей:**
    * `hide.py` — алгоритм внедрения текстового сообщения в младшие биты цветовых каналов пикселей.
    * `seek.py` — алгоритм считывания и декодирования скрытого текста из модифицированного `.bmp` файла.
    * `bleaching.py` — модуль очистки (обесцвечивания/удаления) скрытых данных.

### `lab4/` — Вычислительная геометрия (Множества точек)
* **Что делает:** Интерактивное приложение для решения геометрических задач на плоскости. На основе заданных множеств точек программа находит специфические конфигурации (например, треугольники, окружности или многоугольники), удовлетворяющие экстремальным условиям (максимальная площадь, вхождение точек и т.д.).
* **Особенности:** Интерактивный графический холст `Canvas` с возможностью динамического добавления точек кликом мыши и масштабированием координатной сетки.

### `lab5/` — Компьютерная анимация и симуляция физики
* **Что делает:** Мультимедийное приложение на базе `Pygame`, реализующее плавную цикличную 2D-анимацию сложного объекта (например, механическая модель, вращение, движение планет или колесных систем).
* **Особенности:** Использование тригонометрических трансформаций для пересчета координат движущихся и вращающихся элементов в реальном времени, контроль плавности отрисовки с помощью `pygame.time.Clock`.

### `rk/` — Рубежный контроль
* **Что делает:** Практические задачи повышенной сложности, выполненные в ограниченное время (включая побитовый разбор цвета пикселей, стеганографический анализ и работу со структурами данных).
